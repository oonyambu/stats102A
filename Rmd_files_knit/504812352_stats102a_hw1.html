Flowchart of gcd function

1.2 Pseudocode

START:

INPUT: positive integers a and b

IF y is not a positive integer

  • RETURN Error Message

  • STOP function

ENDIF

SET: a mod b to remainder

SET: minimum of a and b to saved_value

WHILE: remainder >= 1

  • SET: remainder to saved_value

  • SET: b mod remainder to remainder

ENDWHILE

RETURN: saved_value

END

3. is_prime helper function

3.1 Flowchart

flowchart of is_prime function

3.2 Pseudocode

START:

INPUT: vector of positive integers y

SET: NULL to result_vector

SET: 1 to n

FOR every n in length of y IF y is not a positive integer

  • RETURN Error Message

  • STOP function

ENDIF

  • SET: (nth value of y) - 1 to last_num

  • SET: 2 to i

  • FOR every i in last_num

    • SET: (nth value of y) mod i to remainder

    • IF remainder is < 1 and (nth value of y) != 2

    • SET: (nth value of result_vector) to FALSE

    • BREAK out of inner FOR loop

    • ENDIF

    • SET: result_vector[n] to TRUE

    • INCREMENT i by 1

  • END FOR

  • INCREMENT n by 1 END FOR

RETURN Result_vector

END

4. get_factors function

4.1 Flowchart

flowchart of get_factors function

4.2 Pseudocode

START:

INPUT: postive integer y

IF y is not a positive integer

  • RETURN Error Message

  • STOP function

ENDIF

SET: 1 to 2

SET: NULL to result_vector

WHILE y >= i

  • IF y mod i < 1

    • SET: y / i to y

    • SET: append i to result_vector

    • SET: i to 2

  • ELSE

    • INCREMENT i by 1

  • ENDIF END WHILE

RETURN: result_vector
