Problem 1

The gcd function: Flow chart

knitr::include_graphics("hw1_1.jpeg")

Pseudocode

gcd(a,b)

INPUT Two integers a, b  
OUTPUT An integer gcd such that it is the greatest common divider of a and b

Step 1  If a and b are not both positive integers do  
              print the error message "Both of the inputs   need to be a positive integer!"; (Error handling)
              STOP. 
Step 2  While the remainder of max(a,b) divided by min(a,b) is not 0 do    
              set a to the modulus of max(a,b) divided by min(a,b); 
              set b to the remainder of max(a,b) divided by min(a,b); 
Step 3  OUTPUT(b);
        STOP.

The helper functions: The is_primefunction: Flow chart

knitr::include_graphics("hw1_2_is_prime_single.jpeg")

Pseudocode

is_prime_single()

INPUT A number x
OUTPUT A boolean value TRUE or FALSE, corresponding to whether x is a prime number

_Step 1_ If x is an integer greater than 1 do
            initialize and set x_sqrt to the square root of x;
            initialize and set i = 1.
_Step 2_ While i <= x_sqrt do
            if x mod i is equal to 0 do
              OUTPUT(FALSE);
              STOP.
            set i = i+1.
_Step 3_ OUTPUT(TRUE)
         STOP.

Vectorize is_prime_single() and assign it to the name is_prime. is_prime now takes vectors as inputs and outputs vectors consisting of boolean values (True if the element is prime; false if not.)

The get_factor function: Flow chart

knitr::include_graphics("hw1_2_get_factor.jpeg")

Pseudocode

get_factors()

INPUT x (One single number)
OUTPUT factors (a vector containing all the prime factors of x)

Step 1  If x is not an integer do
          print "x must be an integer greater than 1!"; (Error handling)
          STOP.
Step 2  If x is a prime number do
          OUTPUT(x).
Step 3  Initialize an empty vector, factors.
Step 4  Initialize and set i = 2
Step 5  While i <= x do
          if i is a prime number do
            while x mod i equals 0 do
              append i to factors;
              set x to x/i.
          set i to i + 1.
Step 6  OUTPUT(factors);
        STOP.
