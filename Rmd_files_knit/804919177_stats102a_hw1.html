1. Greatest Common Divisor

gcd() Flowchart

gcd() Flowchart

gcd() Pseudocode

Check the inputs for validity and throw an error if either is not a positive integer.
While the two numbers are not divisible by each other,
replace the larger of the two numbers with the remainder after dividing the two numbers.
Repeat this until the new two numbers are divisible by each other.
Once the two numbers are divisible, the smaller number is the gcd, so return that number.

gcd_three() Flowchart

gcd_three() Flowchart

gcd_three() Pseudocode

Take the gcd of two of the 3 integers using the gcd function.
Take the gcd of the third integer with the gcd of the first two integers, and return that number.

2. Prime Factorization

get_factors() Flowchart

get_factors() Flowchart

Pseudocode

Check the input to make sure it is an integer larger than two.
Check if the input is divisible by increasingly large primes -
if it is divisible, divide the input by that prime and save that prime as a prime factor, then continuing dividing the now smaller input by larger primes
Once the input is reduced to 1, you have the complete list of prime factors.

is_prime() Flowchart

is_prime() Flowchart

is_prime() Pseudocode

If the input is less than two or not an integer, it is not prime.
Check if the input is divisible by an integer between 2 and its square root
If the input is divisible, it is not prime
Otherwise, it is prime
